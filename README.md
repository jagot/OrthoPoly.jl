# OrthoPoly

[![Build Status](https://travis-ci.org/jagot/OrthoPoly.jl.svg?branch=master)](https://travis-ci.org/jagot/OrthoPoly.jl)

Generation of a few of the
[orthogonal polynomials](https://en.wikipedia.org/wiki/Orthogonal_polynomials).

Example usage:
```julia
julia> OrthoPoly.H(5)
Poly(120x - 160x^3 + 32x^5)
```
